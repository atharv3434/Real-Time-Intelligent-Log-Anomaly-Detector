# Real-Time AI Log Anomaly Detector 🛡️🤖

An end-to-end MLOps pipeline designed to detect system intrusions and application failures by scanning server logs in real-time using unsupervised Machine Learning (Isolation Forest).

## ✨ Key Features
- **Unsupervised ML Engine**: Built with `scikit-learn` to flag statistical anomalies without manual data labeling.
- **RESTful Serving API**: Served via `FastAPI` capable of low-latency inferences.
- **Production Isolation**: Fully dockerized environment ready for microservice deployments.

## 🛠️ Tech Stack
- **Languages**: Python 3.10
- **Machine Learning**: Scikit-Learn, Numpy, Joblib
- **API Framework**: FastAPI, Uvicorn, Pydantic
- **DevOps**: Docker, GitHub Actions (CI)
